## Правила

- Сначала прочитать последний недельный разбор.
- Затем сверить GitHub Project «Антон — Всё» и свежую приоритизацию.
- В план брать только то, что можно проверить к концу недели.
- Очередь не считать задачей недели без решения Антона.
- Изменения в GitHub делать только после подтверждения.
