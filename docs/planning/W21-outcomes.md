# Недельный план на 21-ю неделю — разворот сервиса

Период: **18–24 мая 2026**.  
Статус: черновик по формату недельного планирования и приоритизации Серёжи Риса.  
Источник: свежий снимок сервиса 15.05.2026, приоритизация разворота сервиса, задачи в Гитхабе.

## Контекст недели

Главная задача недели — перевести стратегию сервиса из «куда расти» в «что уже в понедельник двигает деньги и разгружает операционку».

Ограничение по Антону: рабочее окно нестабильное, ориентир **4–5 часов в день**, поэтому план делится на:

- решения Антона, где нужен владелец бизнеса;
- работу агента и команды, которую можно подготовить без постоянного участия Антона;
- список запретов, чтобы не открыть новое направление раньше кассовой дисциплины.

## Расклад по срочности и важности

### Важное + срочное — Антон делает сегодня/в понедельник

- Утвердить правило предоплаты запчастей для физиков.
- Утвердить минимальный чек и платную диагностику.
- Запретить новые исключения по постоплате без лимита.
- Выбрать, кто делает первые 30 касаний по хвостам.

### Важное + не срочное — запланировать на неделю

- Карта госзаказов.
- Роль координатора/диспетчера.
- Лимиты незавершёнки в ежедневном контуре.
- Отдельный отчёт о прибылях и убытках для проката перед любым новым пунктом.

### Срочное + делегируемое — агент/команда

- Список хвостов и сегментация.
- Тексты сообщений клиентам по деталям, выдаче, хранению.
- Черновик вакансии координатора.
- Шаблон карты контракта.

### Не срочное / не брать

- Новый город.
- Полноценный новый пункт проката.
- Найм мастера первым.
- Большой продукт для компаний и малого бизнеса без кассовой дисциплины внутри сервиса.

## Стоп-условия недели

- Если готовые и старые заказы не разобраны — не обсуждать новую точку проката как ближайший шаг.
- Если правило предоплаты деталей не внедрено — не расширять поток ремонта.
- Если карта госзаказов не собрана — не считать тендеры «ростом», считать их риском дебиторки.
- Если незавершёнка растёт — останавливать новые входы в низком чеке.

- Публичная страница стратегии в ИБ-ленте.
- Задачи Гитхаба: № 221, № 223, № 1 и № 8.
