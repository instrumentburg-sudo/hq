# Недельный разбор

Здесь лежат короткие разборы недели по штабу: что реально закрыто, что зависло, что переносится, что надо решить Антону.

## Формат файла

Файл: `WNN.md`, например `W20.md`.

```markdown

- Осталось в работе:
- Перенесено:
- Отменено:

## Что мешало
-

## Решения Антона
-

## Что агент может сделать сам
-

-
```

## Источники

- GitHub Project «Антон — Всё».
- `tasks/*/*.md` в штабе.
- Коммиты по ключевым проектам.
- CRM-карточки с датами следующих шагов.

Не создавать и не переносить задачи в GitHub без подтверждения Антона. Разбор — это факт и предложение, не самостоятельное управление очередью.
