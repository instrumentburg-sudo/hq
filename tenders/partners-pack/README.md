## Принципы работы

1. **Не конфликт, а процедура.** Все действия = «упорядочиваю документооборот для своей бухгалтерии».
2. **Не показывать карты.** Партнёрам не сообщать о готовящихся допсоглашениях до момента, когда они подписаны.
3. **Не подписывать ничего на встречах.** Любые договорённости — на следующий день после паузы.
4. **Сохранять всё.** Каждое письмо, скрин, чек, трек — в облако.
5. **Не действовать в эмоциях.** 24 часа паузы перед любым письменным ответом партнёру.
