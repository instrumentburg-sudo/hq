# Ежедневная сводка задач HQ через Hermes

## Что сделано
Настроена автоматизация Hermes: каждый день в 10:00 по Екатеринбургу она читает задачи HQ и присылает короткую сводку в Telegram.

## Результат
- Hermes cron job: `daily-hq-task-summary-ekb-1000`
- Расписание: `0 5 * * *` UTC
- Источник: `hq/tasks/*/*.md`
- Сборщик контекста: `~/.hermes/scripts/hq-task-summary-context.py`
- Доставка: личка Антона в Telegram

## Проверка
- Проверочное сообщение пришло.
- После обратной связи формат укорочен до компактной Telegram-сводки.
- Служебная cron-обёртка отключена через `cron.wrap_response=false`.
- Последний ручной прогон завершился со статусом `ok`.

## Заметки
Это первый реализованный вариант практики «Агент-надзиратель по расписанию» из урока 2 Personal Corp.
